# ENQview


## install方法
```r
if (!require(devtools)){
    install.packages('devtools')
    library(devtools)
}
devtools::install_github("419kfj/ENQview", upgrade="never")
```

## 呼び出し手順

- インストールのあと

library(ENQview)

## データを使えるようにする。
```r
data(taste_J)
ENQview(taste_J) # 分析対象データの単純集計
Shiny_speMCA(taste_J) #分析対象データのspeMCA分析
```
